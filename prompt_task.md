# Prompt Engineering Task

This task demonstrates prompt engineering using Role, Task, Context, and Format, along with different chatbot input types.

# Prompt 1 (Text Input)
Role: NLP Engineer  
Task: Build a spam classifier  
Context: Input is text message  
Format: Provide Python code  

Prompt:
You are an NLP Engineer. Build a spam classifier that takes text input and predicts whether it is spam or not. Provide Python code.

# Prompt 2 (Voice Input)
Role: AI Engineer  
Task: Design voice-based chatbot  
Context: Input is speech converted to text  
Format: Explain architecture and tools  

Prompt:
You are an AI Engineer. Design a chatbot that accepts voice input, converts it to text, and responds accordingly. Explain architecture and tools used.

# Prompt 3 (File Input)
Role: Data Analyst  
Task: Analyze dataset  
Context: Input is CSV file  
Format: Provide insights and visualization code  

Prompt:
You are a Data Analyst. Analyze a CSV dataset uploaded by the user and provide insights along with visualization code.

# Prompt 4 (Image Input)
Role: Computer Vision Engineer  
Task: Image classification  
Context: Input is uploaded image  
Format: Provide model and code  

Prompt:
You are a Computer Vision Engineer. Build a model that classifies objects from an uploaded image. Provide model details and Python code.

# Conclusion :
Chatbots can handle multiple input types such as text, voice, images, and files. Prompt engineering helps define how the chatbot should respond effectively.
